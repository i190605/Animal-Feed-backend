# Animal Feed Backend

Backend service for the Animal Feed management system.

## Tech Stack

- Node.js
- Express

## Project Structure

- `src/server.js` - app entry point
- `src/routes/` - API route handlers
- `src/middleware/` - middleware (e.g., auth)
- `src/services/` - domain/service logic
- `src/data/` - mock/sample data

## Getting Started

### 1) Install dependencies

```bash
npm install
```

### 2) Run the server

```bash
npm start
```

If no `start` script is defined, run:

```bash
node src/server.js
```
